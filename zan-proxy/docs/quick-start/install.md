# 安装

## 通过[Yarn](https://yarnpkg.com/)安装

```shell
    yarn global add zan-proxy
```

## 通过[NPM](https://www.npmjs.com)安装

```shell
    npm i -g zan-proxy
```

## 安装检验

```shell
    zan-proxy --version
```
