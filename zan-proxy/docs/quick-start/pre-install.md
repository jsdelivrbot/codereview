# 前置条件

## OpenSSL: Version >= 0.9.8

### 版本查看
```shell
    openssl version
```

### 安装
```shell
    brew install openssl
```

详情可参考[OpenSSL](https://www.openssl.org/)。

## Node: Version >= 6.10.1

### 版本查看

```shell
    node --version
```

### 安装

```shell
    brew install node
```

详情可参考[Node](https://nodejs.org/)。
